# B站视频分享给好友更新日志

## V0.4.8

更新：2026年6月9日

- 支持在番剧播放页分享面板中追加“B站好友”入口。
- 新增番剧 `ep_id` 分集信息解析，番剧页点击入口后可读取当前分集标题、封面和 BV 号。
- 番剧分集私信分享优先发送当前 `bangumi/play/ep` 链接，普通视频页分享逻辑保持不变。

## V0.4.7

更新：2026年6月5日

- 好友标签栏右侧新增“清空所选”按钮，可直接清空当前已勾选用户；没有已选用户时按钮置灰。

## V0.4.6

更新：2026年6月5日

- 点击发送后先展示发送过程，完成后展示逐个用户的发送结果。
- 发送结果底部新增“继续”操作，可回到选择用户面板并清空勾选和搜索状态。
- 支持一次勾选多个用户并顺序发送，单个用户失败不会影响后续发送。
- 单次发送建议最多 5 个用户，选到第 6 个及以上时禁用发送按钮并显示简短超限提示。
- 发送结果中的用户昵称支持点击后跳转到对应 B 站空间。
- 发送结果汇总中的成功和失败数量使用不同颜色区分。
- “最近聊天 / 全部好友”切换时保留两个面板实例，避免重复卸载和重建列表组件。
- 面板改为通过隐藏/显示切换可见性，减少头像资源在切换时重复请求。
- 简化用户列表项点击逻辑，统一选择用户的交互处理。
- 修正“全部好友”关系列表切换时可能自动触发翻页的问题。
- 收紧滚动分页触发条件，避免列表状态变化时重复加载下一页。
- 保持关注、粉丝和搜索列表各自的滚动位置，切换后恢复到之前浏览的位置。
- 修正分享弹窗右上角关闭图标视觉上未垂直居中的问题。

- 统一 WBI 签名混合密钥重排表常量命名。

## V0.4.0

更新：2026年5月30日

- 将分享入口和弹窗 UI 重构为 Preact 实现，保留原有交互和样式。
- 分享弹窗改由 Preact 管理状态，继续支持最近聊天、关注/粉丝切换、搜索、滚动加载和发送结果提示。
- 构建链路接入 Preact/Vite 预设，依赖随 userscript 产物打包。

## V0.3.0

更新：2026年5月29日

- 分享弹窗 UI 拆分为多个组件，便于独立维护弹窗、视频预览、用户列表、搜索框等区域。
- 新增“最近聊天 / 全部好友”切换，默认继续展示最近聊天列表。
- “全部好友”支持在“我的关注 / 我的粉丝”之间切换，并从对应关系列表选择接收人。
- 支持昵称搜索、300ms 防抖和滚动分页加载；粉丝搜索会提示仅筛选已加载用户。
- 切换来源、分类或搜索词时会清空已选用户，避免发送给当前列表外的用户。

- 发送结果会区分“视频卡片发送成功”和“已降级为文本链接发送”。
